# kunle-personal-site

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Starting the Servers

#### Backend Server
```bash
cd backend
npm install
npm start
```
The backend server will run on `http://localhost:4000`

#### Frontend Server
```bash
cd frontend
npm install
npm run dev
```
The frontend server will run on `http://localhost:3000`
