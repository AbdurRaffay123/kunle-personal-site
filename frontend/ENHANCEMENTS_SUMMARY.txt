🚀 HOW TO USE
========================================

Enable Mock Data Mode:
1. Edit .env.local
2. Set: NEXT_PUBLIC_USE_MOCK_DATA=true
3. Run: npm run dev
4. See mock content instantly!

With Backend:
1. Set: NEXT_PUBLIC_USE_MOCK_DATA=false
2. Set: NEXT_PUBLIC_API_BASE_URL=your-api
3. Automatic fallback on errors
